# Netflix Data Analysis

## 📊 Project Overview
Analyzed Netflix dataset using Python to uncover trends and insights.

## 🛠 Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

## 📈 Key Insights
- Movies dominate Netflix content  
- USA is top content producer  
- Content increased after 2016  
- International Movies are most popular
